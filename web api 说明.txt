【响应格式】
{
    "status":"状态"
    "result":"结果"
}

状态：
-1 参数不完整或出现错误
0 未找到
>0 找到且返回结果条数

结果：
返回错误信息
返回结果json




【查找景点】spot-retrieve

{
    "id":"景点id",
    "name":"景点名称",
    "address":"景点地址"
}
(三种条件任意一个即可)

用id查找，找到返回一个结果，找不到返回0
用name或address查找，找到返回数量和结果，找不到返回0
条件缺失返回-1




【查找订单】order-retrieve

{
    "orderid":"订单id",
    "userid":"用户id",
    "spotid":"景点id"
}
(三种条件任意一个即可)

用id查找，找到返回一个结果，找不到返回0
用spotid或userid查找，找到返回数量和结果，找不到返回0
条件缺失返回-1




【创建订单】order-create

{
    "userid":"用户id",
    "spotid":"景点id",
    "count":"购买数量"
}
(须满足三个条件)

成功返回1
找不到資料或条件缺失返回-1




【检查用户是否注册】user-check

{
    "lineid":"line id"
}

已注册返回1
未注册返回0
